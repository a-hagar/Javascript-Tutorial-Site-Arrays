What are arrays?

Arrays are a list of values, including strings, numbers, booleans, or other variables into a single variable. When you need a items from a list of common items, it is easier to have them combined into an array than having inidividual variables.

How to write an array?

Arrays can also be written in individual variables with an assigned numbered indexes.

Arrays always start at 0, so remember that your first will start at 0 instead of 1.

Array methods

There are several ways to manipulate an array. Here are the list of some Javascript array methods

array.join(,)

The join function turns the array items into a string. The brackets hold the separator that goes between each new string, such as a comma or "and."

array.sort & array.reverse

These two functions organize the arrays. The sort function organizes in alphabetical order, and the reverse function sorts in descending order.Warning: this will not work correctly with numbers, as they will be organized by their first character (ex. 10, 2, 351, 42)

array.splice(startIndex, totalDelete, newItem)

The splice function adds and removes any item from the middle of an array. You have to state in the array you will delete (startIndex), how many indexes you plan to delete (totalDelete), and any new items you wish to add (newItem).

array.slice(startIndex, endIndex)

The slice function is similar to splice in removing pieces of an array, but it does not alter the array as the splice function does. Slice creates a new array with the new addition included.

array.indexOf(itemName)

If you have an array item and want to find its index number in the array, use the indexOf function.

array.pop() & array.push()

The pop and push functions serve similar functions to add and remove items.

array1.concat(array2)

The concat function combines the two arrays together. The second array variable always goes into the bracket.

array.shift & array.unshift

The shift and unshift functions manipulate the first item of the array. Shift removes the first item, whereas unshift adds a new item to the front of the array.
